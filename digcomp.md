# Digital skills — DigComp (base solida e “testabile”)

Per Digital Skills la strategia migliore è studiare con un framework UE:
**DigComp (Digital Competence Framework for Citizens)**.

Pubblicazione DigComp 2.2 (Publications Office):  
https://op.europa.eu/en/publication-detail/-/publication/50c53c01-abeb-11ec-83e1-01aa75ed71a1/language-en

## 1) Le 5 aree (da sapere a memoria)
1) Information & data literacy  
2) Communication & collaboration  
3) Digital content creation  
4) Safety  
5) Problem solving  

## 2) Esempi “da quiz” (schemi)
### Safety
- phishing, MFA, password manager
- aggiornamenti, backup

### Data literacy
- affidabilità fonti
- interpretazione grafici

### Collaboration
- netiquette
- documenti condivisi (versioni)

### Content creation
- copyright/licenze
- editing e formati

### Problem solving
- troubleshooting
- scegliere strumento giusto

## 3) Metodo
Per ogni concetto: definizione + 3 segnali + 3 azioni corrette → 3 MCQ.
